# insitu_data

This repository includes the match-ups of HyperNav/SGLI (sgli_hypernav_matchup_v4.csv is the most recent one) and in-situ Rrs data obtained during the SOKOWASA cruise (SOKOWASA_HyperPro_Rrs_with_date_time_v2.csv).Files were updated including lat/lon and time info.

-version 3 file includes HyperNav/SGLI matchups up to **Mar 2025**.

-version 4 file includes HyperNav/SGLI matchups up to **Nov 2025**.

*Note that HyperNav data has been reprocessed. Therefore, it is normal to see in-situ Rrs differences in different versions of data.
